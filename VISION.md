# 🌌 Axiom 2.0: The Neural-Spatial Interface

> **Status:** Unrestricted Production Vision
> **Aesthetic:** Dark Matter Glass, Liquid Geometry, & Generative Space
> **Core Paradigm:** Evolve from a "clinical tool" into a symbiotic, high-performance command nexus for autonomous agents and BYOM (Bring Your Own Model) orchestration.

## 1. 🖋️ Cognitive Typography (The "Living Font" Kit)

We are moving past static fonts. Typography in Axiom 2.0 uses **Variable Font Axes** connected to system telemetry. The text reacts to the data it represents.

| Layer | Font Choice | The "Cognitive" Factor |
| :--- | :--- | :--- |
| **Headers** | **Clash Display (Variable)** | The font weight subtly shifts based on the "importance" or compute-cost of the current view. Heavy operations get bolder, commanding space. |
| **UI Body** | **Geist (Vercel)** | Perfect geometric clarity. But on hover, characters briefly undergo a subtle typographic "decryption" animation, giving a matrix-like feel to mundane text. |
| **Inference/Code** | **Berkeley Mono** | When a model is actively generating tokens or streaming data, the text doesn't just appear—it "burns in" with a micro-millisecond white glow before cooling to its syntax color. |
| **System Telemetry** | **JetBrains Mono** | Locked rigidly to a monospaced grid, ensuring that rapidly fluctuating numbers (like GPU temp or token/sec rates) never cause layout jitter. |

## 2. 💎 "Generative" UI Innovations

### A. The BYOM "Nexus Canvas" (Replacing the Query Block)
Flat boxes are dead. The workspace is now an infinite, zoomable spatial canvas.
* **Visuals:** Workspaces are defined by **Volumetric Fog borders** rather than hard lines, using WebGL.
* **The Model Node:** When you plug in a custom model, it appears as a pulsing, slow-rotating geometric core.
* **Agentic Routing:** When agents communicate, the UI draws ephemeral, glowing "fiber optic" splines between them. You don't just see the logs; you watch the data physically traverse the screen.

### B. "Liquid Metal" State Transitions
Eliminate all standard buttons for critical actions.
* **The Override/Execute Action:** Instead of a button, it's a **magnetic interaction zone**. As the cursor approaches, the button physically warps and pulls toward the mouse.
* **Shader Material:** Uses a custom GLSL shader. When clicked, it ripples like mercury, and a shockwave distorts the pixels of the entire UI underneath it for a fraction of a second.

### C. Latent Space Projection (The Ghost Preview 2.0)
Why wait for the query or prompt to finish?
* **Innovation:** As the user types, the UI predicts the required visualization. Before the data arrives, a wireframe "hologram" of the chart or table constructs itself.
* **Visual execution:** It looks like a 3D blueprint being scanned by lasers. As the actual JSON/data payload hits the frontend, the blueprint instantly solidifies into the final rich component.

## 3. 🎨 The "Event Horizon" Palette

Say goodbye to `#000000`. We use layered darkness to create infinite depth.

* **The Void (Background):** `#010204` with a subtle, continuously shifting noise grain to prevent it from feeling dead.
* **Primary Energy (Accents):** `#00FFAA` (Hyper-Mint) and `#FF0055` (Plasma Red for critical alerts).
* **The Glass (Surfaces):** `rgba(255, 255, 255, 0.01)` but layered with `box-shadow: inset 0 1px 0 rgba(255, 255, 255, 0.05)` to catch virtual light on the top edge of every component.

## 4. ⚡ Synaptic Interactions (The "Feel")

* **Gaze & Focus Simulation:** Elements slightly tilt (3D transform) based on mouse position, creating a parallax effect. The UI feels like it's a physical layer of glass floating 2 inches behind your monitor.
* **The Omni-Command (Cmd+K):** It doesn't just open a menu; it **shatters the current view**. The background dramatically blurs and desaturates, and the command prompt materializes from a cluster of geometric shards.
* **Haptic Sound Design:** Pair the visuals with ultra-minimal web audio API triggers. A sub-bass 'thump' on major executions, and soft, glass-tapping sounds when navigating menus.
